# GBAx
GBAx is an open-source Game Boy Advance emulator with a built-in game launcher and library management, based on mGBA. -- Built with Qt

## 🚀 Version 2.0 — Launcher Experience Upgrade

GBAx evolves into a complete Game Boy Advance hub with a powerful built-in launcher and a polished, controller-friendly experience.

### 🎮 Full Game Launcher
- Browse your installed library with a modern, console-style interface  
- Dedicated sections for Installed, Favorites, Recently Played, Browse Games, and Settings  

### ⚡ Faster Startup & Smart Caching
- Significantly improved launch times using optimized library caching  
- Instant loading on subsequent launches  

### 🖼 Automatic Artwork & Visual Library
- Game covers are downloaded automatically for a richer browsing experience  
- Improved artwork handling and smoother rendering across the launcher  

### 🔁 Seamless Launcher ↔ Emulator Flow
- Launch games instantly and return back to the launcher without disruption  
- Integrated fullscreen and emulator control behavior  

### 🎯 Controller Navigation Support
- Navigate the entire launcher using a controller  
- Built-in shortcuts for launching, switching, and exiting games  

### ⭐ Favorites & Continue Playing
- Mark games as favorites for quick access  
- Resume your last played game instantly from the Continue Playing panel  

### ⚙️ Improved Settings & Customization
- Cleaner settings layout with better organization  
- Artwork management, library controls, and performance improvements  

### ✨ UI & Performance Refinements
- Redesigned interface focused on clarity, speed, and usability  
- Smoother scrolling, better layout balance, and improved responsiveness
